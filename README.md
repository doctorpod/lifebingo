# Life Bingo

Make your life better by incremental habit tweaks inspired by bingo-style boards.

## Technology

- Node JS
- Express
- Bootstrap
- Stimulus

## Setup

Set env vars `NODE_ENV` and `DATABASE_URL`.

Create database:

    sequelize db:create

Seed database (optional):

    sequelize db:seed:all

## Run

Run locally:

    npm run dev

Build and run in docker locally:

    npm run docker
